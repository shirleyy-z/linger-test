# Linger — Increment 2

Linger is a private, nostalgic memory scrapbook. Increment 2 contains the working authentication foundation plus a complete personal memory timeline with private media uploads, backdating, reminders, editing, search, favourites, deletion, and simple resurfacing.

## Run locally

```powershell
npm install
npm run dev
```

Read `SETUP_INCREMENT_2.md` before starting. Run `supabase/increment_2.sql` once in the Supabase SQL Editor after the Increment 1 database setup.

## Current roadmap

- Increment 1: authentication, protected dashboard, visual foundation
- **Increment 2: personal memories, uploads, timeline, reminders**
- Increment 3: shared collections and Gemini memory intelligence
- Increment 4: Wrapped, polish, deployment, and presentation flow


## Increment 3

Adds global in-app reminder modals, built-in voice recording, dedicated diary and letter editors, and additional scrapbook motifs. Run `supabase/increment_3.sql` after Increment 2.
